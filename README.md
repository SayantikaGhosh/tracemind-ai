# TraceMind AI

Production incident root cause analysis powered by 
Cognee knowledge graphs.

## What It Does
When your application crashes, TraceMind finds the 
root cause automatically by connecting code deployments,
metric alerts, and team conversations into a causal chain.

## Tech Stack
- Cognee Cloud (knowledge graph memory)
- Groq API — Llama 3.1 8B (RCA report generation)
- FastAPI (backend)
- PostgreSQL + pgvector (raw events + vector search)
- React + Tailwind (frontend)
- D3.js (causal graph visualization)
